# 跨框架渲染模式对照

## 综合论点

**六种渲染模式中，SSG/SSR/Streaming 全框架通用；ISR 思想通用但术语各异；RSC/RCC 为 React/Next 精确二分，其他框架以同构或 Islands/`client:*` 达成类似效果**——选型时对照本表即可 API 映射，无需重学概念。

## 能力矩阵

来源：[[Web 渲染架构：六种模式与跨框架对照]]

| 模式 | [[Next.js]] | [[Nuxt]] | [[SvelteKit]] | [[Astro]] | Rails/Django |
|------|------------|---------|--------------|----------|--------------|
| **[[SSG]]** | `generateStaticParams` | `nuxt generate` | `prerender` | 默认 SSG | 不适用 |
| **[[SSR]]** | `dynamic = 'force-dynamic'` | `ssr: true` | `+page.server.js` | Server Islands | 原生默认 |
| **[[ISR]]** | `revalidate` | swr / isr | invalidate + CDN | 按需渲染 | 需配合 CDN |
| **[[Streaming SSR]]** | Suspense + `loading.tsx` | Suspense 流式 | stream | 渐进式渲染 | Channels 等 |
| **[[React Server Components]]** | App Router 默认 | 无（同构） | 无（同构） | Server Islands | 不适用 |
| **[[React Client Components]]** | `'use client'` | 默认同构 | 默认同构 | `client:*` | 不适用 |

## 行业通用性分级

| 模式 | 通用性 | 说明 |
|------|--------|------|
| SSG / SSR / Streaming | ⭐⭐⭐ | 早于 React 存在；PHP/Rails 即 SSR |
| ISR | ⭐⭐ | 术语 Vercel 化；SWR 为 CDN 通用 |
| RSC / RCC | ⭐ React | 思想被 Astro/Fresh 借鉴 |
